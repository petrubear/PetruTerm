## Technical Reference
- Shell exit: `Event::ChildExit(i32)` — alacritty_terminal 0.25.1
- EventLoopProxy: wakes NSApp immediately; `user_event()` handles it
- Custom title bar: `HasWindowHandle → AppKitWindowHandle.ns_view → [view window]`
  + `setStyleMask|(1<<15)`, `setTitlebarAppearsTransparent:YES`,
    `setTitleVisibility:1`, `setMovableByWindowBackground:YES`
- Working dir: `dirs::home_dir()` → `PtyOptions::working_directory`
- Icon regen: `swift scripts/gen_icon.swift assets/AppIcon.png && ./scripts/bundle.sh`
- Bundle: `./scripts/bundle.sh` → `dist/PetruTerm.app`
